# RN-StarWars
1. `cd` to project root (`cd reactnativestarwars`)
2. `yarn`
3. `cd ios`
4. `pod repo update`
5. `pod install` 
6. `cd ..`
7. `npx react-native start`
8. In a separate terminal
  - For ios simulator, run `npx react-native run-ios`. 
  - For android emulator, run `npx react-native run-android`.
